# Terraria Fishing Bot

Terraria advanced fising bot writted fully on Rust. Made to practise/learn Rust.

## Features

- [x] Reliable fishing in water/lava/honey
- [ ] Overlay
- [x] Automatic potion use
- [ ] Fast unload to nearbay chests
- [ ] Handle events (Bloodmon, Invesions)
- [ ] Working with sonar potion
- [x] Saving settings

## Manual

1. Install Terraria-Fishing-Bot.exe and run it.
2. Prepare position for fish where no moving entities are.
3. Setup cursor just slightly below where you want to fish.
4. Press Q to start and don't move both cursor and your character.
5. Press R to stop (If bot just catch and cast).
6. Press P to exit.

# Things to Know

- The bot `WILL NOT WORK` if you do not have waves enabled.
- Bot tracking area around cursor.
- The bot may not work properly in hell due to distortion. It is better to set a contrasting background.
- During heavy rain, false alarms sometimes occur. It is better to put up a roof.
- Bot working on 5 fps which seems to be enough.
